Classification of programming languages

Programming languages can be classified in terms of their level into low-level programming languages and high-level programming languages, and the level here is meant by the extent to which the language is stripped and easily understood by humans, so the most abstract languages are machine language Machine code is a low-level programming language, and the closer the programming language is to the machine language, the lower the level, since the machine language followed by the assembly language is the lowest in the level. In machine language, commands are written as Directly in the language that the computer understands, namely the numbers, and in the assembly language, the commands are written in a slightly simpler way than the machine language, so that a specific phrase indicating the command is written instead of writing it in numbers, as a phrase (MOV AL, 61h) for example.
